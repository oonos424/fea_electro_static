# Static Electrodynamics Solver with Finite Element Approximation

A finite element method (FEM) solver for static electrodynamics problems, intended as a stepping stone toward a full CFD solver.

## Project Structure

```
fea_electro_static/
├── doc/           # Documentation and LaTeX files
├── src/           # Source code
├── tests/         # Unit and integration tests
├── examples/      # Example problems and demonstrations
├── data/          # Input data files
└── results/       # Output results and visualizations
```

## Description

This project implements a finite element approximation solver for static electrodynamics problems. It serves as a foundation for developing a more comprehensive CFD solver in the future.
